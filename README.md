# To-Do List App 📝

A simple Android application for managing daily tasks. Built with **Java** and **Android Studio**.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| ✅ Add Tasks | Enter new tasks via text input |
| ✅ Confirmation Dialog | Confirm before adding |
| ✅ Toast Notifications | Visual feedback for actions |
| ✅ Clean UI | Simple and intuitive interface |

---

## 🛠 Tech Stack

| Technology | Version |
|------------|---------|
| Language | Java |
| Minimum SDK | API 25 (Android 7.1 Nougat) |
| Target SDK | API 33 |
| Build System | Gradle |
| IDE | Android Studio (Panda 2) |

---


## 🚀 How to Use

| Step | Action | Result |
|------|--------|--------|
| 1 | Enter task in text field | Text appears |
| 2 | Click **ADD** button | Dialog opens |
| 3 | Click **Yes** to confirm | Task added (toast) |
| 4 | Click **No** to cancel | Dialog closes |

---

## 🚀 How to Run the Project

1. Open **Android Studio**
2. Click **File → Open**
3. Select this folder (`ToDoList`)
4. Wait for **Gradle** to sync
5. Click **Run** ▶️
